**Завдання 4:**

1️. Підготовка користувачів та груп

У зв’язку з тим, що мій користувач developer\_user не має root-доступу, я не змогла створити реальних користувачів та груп.

Теоретично, для цього використовуються такі команди:

\# Створення груп

sudo groupadd tech

sudo groupadd dev

sudo groupadd fin

sudo groupadd founders

sudo groupadd guests

\# Встановлення shell для альтернативних командних інтерпретаторів

sudo dnf install zsh \-y

sudo dnf install fish \-y

\# Створення користувачів

sudo useradd \-m \-g tech \-s /bin/bash tech1

sudo passwd tech1

sudo useradd \-m \-g dev \-s /bin/zsh dev1

sudo passwd dev1

sudo useradd \-m \-g founders \-s /usr/bin/fish founder1

sudo passwd founder1

sudo useradd \-m \-g fin \-s /sbin/nologin fin1

sudo passwd fin1

sudo useradd \-m \-g guests \-s /sbin/nologin guest1

sudo passwd guest1

Пояснення:

| Група | Користувачі | Shell / доступ |
| :---- | :---- | :---- |
| Technical support | tech1 | bash (повний доступ) |
| Developers | dev1 | zsh (розширені можливості) |
| Founders | founder1 | fish (зручний shell для керівництва) |
| Financiers | fin1 | /sbin/nologin (немає доступу) |
| Guests | guest1 | /sbin/nologin (немає доступу) |
